# System-Programming-Term-Project

# Pre requirements

- Ubuntu OS
- Docker and Docker compose installed
- Git

# Setup Instructions

1. Clone the Repository:
git clone: https://github.com/ulask8139/System-Programming-Term-Project.git

2. Navigate to Project Directory:

3. Build and Start Containers:

docker-compose up --build -d 

4. Verify Services:
Use docker ps to ensure both db and app containers are running.
